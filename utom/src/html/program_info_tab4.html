공통과정과 개별과정 학습이 끝나면 반드시 평가를 진행합니다.
빠른 진도보다 중요한 것은 정확한 진단입니다.

공통 과정

  1. 단원종합 TEST
    (주간평가)

    한 단원의 완성도를
    평가합니다.

  2. 월말평가
    (월간평가)

    2 ~ 3 단원의 완성도를 평가하고
    Clinic과 추가학습을 진행합니다.

  3. 학력진단평가 P/T
    (분기평가)

    분기별로 공통과정에 대한
    평가를 진행합니다.

U-TES (U2M Total Evaluation System) : 과정 I

  • 표지

  • 학습정보

  • 성취도 종합분석

  • 난이도별 종합분석

  • 성취도 상세분석

  • 주간평가 상세분석

  • 월말평가 상세분석

  • 월말평가 분석

개별 과정

  1. 단원종합 TEST
    (주간평가)

    한 단원의 완성도를
    평가합니다.

  2. 학력진단평가 P/T
    (분기평가)

    개별과정의 진도를 마무리하고 해당 학기에 인증평가를 진행합니다.

U-TES (U2M Total Evaluation System) : 과정 II

  • 난이도별 종합분석
